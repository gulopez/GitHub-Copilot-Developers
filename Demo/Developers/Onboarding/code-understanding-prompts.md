# Sample Prompts for Code Understanding & Onboarding

## Explain Unfamiliar Code
- What does this module do?
- Walk me through this function step-by-step.
- Can you explain the purpose of this class?
- What are the main responsibilities of this file?

## Trace Data Flow
- Where does `userId` originate and how is it validated before DB writes?
- Show how data moves from the API layer to the database.
- What transformations are applied to the input before saving?

## Summarize Changes
- Summarize what this PR/commit does and what risks it introduces.
- What are the key changes in this pull request?
- Are there any breaking changes or potential side effects?

---
Use these prompts to accelerate onboarding, clarify code intent, and ensure safe collaboration.
