# Sample Prompts for Communication

## Drafting Replies
- Draft a polite reply to this issue, thanking the reporter and asking for more details.
- Write a response to a pull request, requesting changes in a constructive way.
- Suggest a closing comment for a resolved issue.

## Suggestions for Clear Communication
- Rephrase this message to be more concise and professional.
- Make this feedback more encouraging and actionable.
- Suggest a way to clarify this technical explanation for a non-expert.

## Generate Release Notes
- Summarize the changes from these merged PRs into release notes.
- Create a changelog entry for the latest release.
- List the key features and bug fixes included in this version.

---
Use these prompts to improve communication, foster collaboration, and keep your project documentation clear and up to date.
