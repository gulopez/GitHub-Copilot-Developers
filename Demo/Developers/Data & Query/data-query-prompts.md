# Sample Prompts for Data & Query Work

## Write/Optimize SQL
- Write a SQL query to fetch active users with their last login date.
- Optimize this query for better performance and suggest indexes.
- Rewrite this query to reduce joins and improve speed.
- Analyze this explain plan and recommend improvements.

## Migrations
- Generate a schema migration to add a new column to the users table.
- Create a rollback strategy for this migration.
- Write a migration script to rename a table and update references.
- Suggest best practices for zero-downtime migrations.

## ETL Scripts
- Write a script to clean and validate imported CSV data.
- Generate an ETL pipeline for reconciling transactions between two systems.
- Add data validation and reconciliation checks to this ETL process.
- Suggest improvements for error handling in this ETL script.

---
Use these prompts to accelerate data work, ensure robust migrations, and improve query and ETL performance.
