# Sample Prompts for Developer Productivity & Learning

## Create “How-To” Snippets
- Show a code snippet for logging in this stack.
- Provide an example of using feature flags in this project.
- How do I configure environment variables securely?
- Give a template for setting up application config files.

## Explain Errors and Concepts
- Explain this error message and how to fix it.
- Why did this code fail, and how can I avoid similar issues?
- Describe the concept of dependency injection in this framework.
- What are common pitfalls with async/await in JavaScript?

## Practice/Exercises
- Generate a small kata to practice using Python list comprehensions.
- Create an exercise to learn the basics of React hooks.
- Give me a task to practice writing SQL joins.
- Suggest a hands-on challenge for learning error handling in Go.

---
Use these prompts to boost productivity, deepen understanding, and accelerate learning in your development workflow.
