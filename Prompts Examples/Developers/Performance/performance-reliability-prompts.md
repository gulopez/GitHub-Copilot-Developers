# Sample Prompts for Performance & Reliability


## Find Hotspots and Optimizations
```markdown
Identify performance bottlenecks in this codebase.
```
```markdown
Suggest ways to reduce memory allocations in this function.
```
```markdown
How can I optimize these database queries?
```
```markdown
Where should I add caching to improve response times?
```

```markdown

```
```markdown
How can I avoid race conditions in this code?
```
```markdown
Which concurrency primitives are best for this workload?
```
```markdown

```

```markdown
Add retries with exponential backoff to this API call.
```
```markdown
Implement a circuit breaker for this external service integration.
```
```markdown
How do I use idempotency keys to prevent duplicate processing?
```
```markdown
Where should I add timeouts to improve reliability?
```

Use these prompts to enhance system performance, ensure safe concurrency, and build resilient, reliable applications.
