# Data Generation Instructions Demo

This document demonstrates various prompt and data generation tasks, inspired by the format of Copilot features demos.

---

## Project Planning

### Zero-shot
```
Can you help me create 10 epics, user stories and tasks for an ticketing system?
```


```
Can you help me create 10 epics, user stories and tasks for an ecommerce site?
```


```
can you show the epics, user stories and task in table format?
```


```
can you create a file with this data called "tasks.csv"
```
